# Apple Podcasts Reference Analysis

Source: local Apple Podcasts design rollup, based on a November 2025 Mobbin iOS capture.

## Adapt

- Strong native page titles with direct content below, minimal explanation.
- Persistent bottom navigation with a compact mini player above the tab bar.
- Library organization that uses simple icon rows for destinations and clean two-column artwork grids only where browsing benefits from covers.
- Episode/story rows with title, short metadata, progress or duration, play affordance, and overflow actions.
- Full-screen player with artwork-led color mood, large transport controls, scrubber, sleep timer, playback speed, transcript, queue/source actions, and a close affordance.
- Empty states that are centered, quiet, and actionable without sounding like placeholder text.
- Menus and sheets for secondary actions rather than exposing every option on primary screens.

## Avoid

- Apple Podcasts purple branding, Apple account copy, subscriptions, marketplace discovery, ratings, charts, and promotional upsells.
- Light-mode first composition. Sleepy History should keep the modern hierarchy but make the scene genuinely nighttime.
- Category recommendation prompts and show-following language that do not fit a private generated-story library.
- Recreating Apple Podcasts layouts so exactly that Sleepy History feels like a clone.

## Sleepy History Translation

- Home becomes a personal listening dashboard: continue, generate, recent stories, and generation status.
- Create replaces Apple Podcasts New/Search as a first-class tab because story generation is the core action.
- Library focuses on generated stories, downloads, completed, in-progress, and failed jobs.
- Bookmarks maps to Apple Podcasts Saved, but keeps Sleepy History language.
- Profile becomes an operational surface: enrollment, provider health, privacy, playback defaults, local data.
- Story detail must expose generated transcript, sources, notes, download state, and playback progress with the same seriousness as audio playback.
